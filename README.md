# 🚀 TaskSphere - Emerald Productivity Suite

TaskSphere is a high-performance, full-stack task management application designed with a premium "Emerald Glass" aesthetic. It leverages a modern tech stack to provide a seamless, real-time organizational experience.

![TaskSphere Demo](./demo.gif) 

---

## ✨ Features

- **Full CRUD Workflow:** Create, view, edit, and delete tasks with instant UI updates.
- **Advanced Categorization:** Create custom categories on the fly to organize different aspects of your life.
- **Premium Emerald UI:** A state-of-the-art dark mode interface featuring glassmorphism effects and vibrant emerald accents.
- **Cutting-Edge Tech:** Built with **Tailwind CSS v4** and **React 19**, showcasing early adoption of the latest industry standards.
- **Smooth Animations:** Fluid transitions and modal interactions powered by **Framer Motion**.
- **Robust Backend:** High-speed API endpoints built with **FastAPI** and **SQLAlchemy**.

---

## 🛠️ Tech Stack

### Frontend
- **React 19** (Vite)
- **Tailwind CSS v4** (Utility-first styling engine)
- **Framer Motion** (Production-ready animations)
- **Lucide React** (Consistent iconography)
- **Axios** (Asynchronous API handling)


### Backend
- **FastAPI** (High-performance Python framework)
- **SQLAlchemy** (ORM for database management)
- **Pydantic** (Strict data validation)
- **SQLite** (Efficient local data persistence)
- **Uvicorn** (ASGI server)
